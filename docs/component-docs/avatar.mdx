# Avatar
사용자를 나타내기 위한 폴백(fallback)을 갖는 이미지 요소입니다.

<Tabs>
<TabItem value="js" label="index.js">

```jsx
import React from 'react';
import * as Avatar from '@radix-ui/react-avatar';
import './styles.css';
const AvatarDemo = () => (
 <div style={{ display: 'flex', gap: 20 }}>
   <Avatar.Root className="AvatarRoot">
     <Avatar.Image
       className="AvatarImage"
       src="https://images.unsplash.com/photo-1492633423870-43d1cd2775eb?&w=128&h=128&dpr=2&q=80"
       alt="Colm Tuite"
     />
     <Avatar.Fallback className="AvatarFallback" delayMs={600}>
       CT
     </Avatar.Fallback>
   </Avatar.Root>
   <Avatar.Root className="AvatarRoot">
     <Avatar.Image
       className="AvatarImage"
       src="https://images.unsplash.com/photo-1511485977113-f34c92461ad9?ixlib=rb-1.2.1&w=128&h=128&dpr=2&q=80"
       alt="Pedro Duarte"
     />
     <Avatar.Fallback className="AvatarFallback" delayMs={600}>
       JD
     </Avatar.Fallback>
   </Avatar.Root>
   <Avatar.Root className="AvatarRoot">
     <Avatar.Fallback className="AvatarFallback">PD</Avatar.Fallback>
   </Avatar.Root>
 </div>
);
export default AvatarDemo;
```

## <Highlight color='#1C5AEC'>설치</Highlight>
커맨드라인에서 컴포넌트를 설치하세요.
```bash
npm install @radix-ui/react-avatar
```

## <Highlight color='#1C5AEC'>구성</Highlight>
모든 파트를 가져와서 조합하세요.
```jsx
import * as Avatar from '@radix-ui/react-avatar';
export default () => (
 <Avatar.Root>
   <Avatar.Image />
   <Avatar.Fallback />
 </Avatar.Root>
);
```

### <Highlight color='#1C5AEC'>Fallback</Highlight>
이미지가 로드되지 않았을 때 렌더링하는 요소입니다. 
이는 로드 중이거나 오류가 발생한 경우를 의미합니다. 
로드 중에 플래시가 나타나면 렌더링을 지연시키는 `delayMs` prop을 제공하여 
연결 속도가 느린 사용자만 렌더링할 수 있습니다.
